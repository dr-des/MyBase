///// Учет заправок /////
/////////////////////////
Цели:
Создать базу данных для учета количества заправок

Структура БД:
  таблицы:
    Customer  - клиенты
    Zip       - комплектующие
    Cartridge - основная таблица
  состав таблиц:
    Customer:
      id    - ID клиента (ключ, числовое, автоувеличение)
      name  - Имя клиента/название организации (строка)
      phone - телефон (строка)
      info  - дополнительная информация/электронный адрес (строка)
    
    Zip:
      id    - ID комплектующей
      type  - тип комплектующей
      name  - название
      info  - дополнительная информация
     
    Cartridge:
      id        - ID
      check     - номер заявки
      customer  - ID клиента
      zip_id    - ID комплектующей
      zip_count - количество
      date      - дата выполнения работ
